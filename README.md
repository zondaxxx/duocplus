# duocplus

Тренажёр в стиле Duolingo для подготовки к экзамену по C++ / ООП за неделю.

**310 упражнений · 40 уроков · 8 дней · 45 экзаменационных билетов**

## Что внутри

- 🛤️ **Карта-путь** как в Duolingo: 7 дней по темам курса + бонусный день из интернет-материалов
- 🃏 **5 типов заданий**: выбор ответа, верно/неверно, «что выведет код» (с подсветкой синтаксиса), заполнение пропусков, пары, порядок
- ❤️ Сердечки, XP, стрик, повтор ошибочных вопросов в конце урока
- 📖 Конспект перед каждым уроком
- 📝 Разбор 4 типовых экзаменационных задач («найди ошибку / что выведет»)
- 🎓 Пробный экзамен: 20 случайных вопросов без сердечек
- 📋 Чеклист всех 45 экзаменационных вопросов
- 💾 Прогресс сохраняется в браузере (localStorage), работает офлайн

## Программа

| День | Тема |
|---|---|
| 1 | Сборка, Make/CMake, парадигмы, принципы ООП, классы и friend |
| 2 | Конструкторы, правило 3/5/0, explicit, static, перегрузка операторов |
| 3 | Наследование, ключи доступа, проблема ромба |
| 4 | Полиморфизм, vtable, виртуальный деструктор, абстрактные классы |
| 5 | Шаблоны, специализация, зависимые имена |
| 6 | Метапрограммирование, SFINAE, концепты, RAII, умные указатели |
| 7 | STL + разбор экзаменационных задач |
| 8 | Бонус: приведение типов, deque/map изнутри, алгоритмы, лямбды, SOLID, тестирование |

## Запуск

Никаких зависимостей и сборки:

- **Просто открыть** `index.html` в браузере (двойной клик), **или**
- `python3 serve.py` → открыть `http://localhost:8431` (с телефона — `http://<IP-компа>:8431` в той же Wi-Fi), **или**
- включить **GitHub Pages** (Settings → Pages → Deploy from branch → main, `/ (root)`) → сайт будет на `https://<user>.github.io/duocplus/`
